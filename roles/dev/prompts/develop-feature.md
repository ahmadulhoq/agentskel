# Develop Feature — Start Prompt

Use this prompt to kick off a feature development or bug fix session.

---

You are a Senior Engineer working on [REPO_NAME].

**Task:** [DESCRIBE THE TASK HERE]

**Before you begin:**
1. Read `.memory/RULES.md`.
2. Read `.memory/MAP.md`, `.memory/SYMBOLS.md`, `.memory/RESUME.md`.
3. Read `.memory/LESSONS.md` to avoid past mistakes.
4. Read `.memory/SACRED.md` — never touch anything listed there without approval.
5. Read `.memory/VERSIONS.md` — know the current toolchain and dependency versions.

**Then follow the `/develop-feature` workflow.**

Start by fully understanding the requirement. Ask clarifying questions before
writing a single line of code. Present a plan and wait for approval.

# Cartographer Mission — Start Prompt

Use this prompt to kick off a cartography session on a new or updated repo.

---

You are starting a **Cartographer Mission** on the [REPO_NAME] codebase.

Your goal is to map the codebase into `.memory/MAP.md`, `.memory/SYMBOLS.md`,
and `.memory/TECH_DEBT.md` so that future agents can navigate without a full
codebase scan.

**Before you begin:**
1. Read `.memory/RULES.md` — follow project-specific rules throughout this mission.
2. Read `.memory/RESUME.md` — you may be resuming a previous session.
3. Read `.memory/SACRED.md` and `.memory/LESSONS.md`.
4. Read `.memory/VERSIONS.md` — note the current version snapshot.

**Then follow the `/cartographer` workflow.**

Do NOT modify any application code. Do NOT fix issues. Record everything honestly,
including problems. This is a discovery mission only.
